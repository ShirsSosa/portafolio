Cinelandia

Web de películas infantiles

Cinelandia fue desarrollada en la Hackathon interna de Laboratoria.

Cinelandio nació bajo la necesidad de encontrar películas con contenido apto para niños.

Es así que usamos el API de OMDB (The Open Movie Database) para obtener la data de las películas, realizamos un prototipo de baja fidelidad, el cual testeamos, desarrollamos el prototipo en Figma y construimos nuestra página web de películas para niños.
